# Career Advisor System (C++ Console Application)

A console-based **Career Advisor System** developed using **C++**. This application helps users explore suitable career paths based on their interests, skills, and educational background.

The system provides career information, required skills, and career roadmaps for various professions. Users can also save their favorite career recommendations for future reference.

This project demonstrates fundamental **C++ programming concepts**, including functions, conditional statements, loops, file handling, and basic data management through a practical career guidance application.

---

## Features

* Answer interest-based questions to explore suitable career paths.
* Browse career fields with their required skills and learning roadmaps.
* Match careers based on users' skills and educational background.
* Save favorite career recommendations.
* View previously saved career recommendations.
* Store and manage career information using text files.
* Interactive menu-driven console interface.
